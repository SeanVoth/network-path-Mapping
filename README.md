# Network Drive Mapping (Powershell) 
Helps map known drives quickly for fresh installs or automation rollouts 


<h1>Description</h1>
Project consists of a simple Powershell script that will Map known network drives and assigns the letter.
<br />


<h2>Languages and Utilities Used</h2>

- <b>Powershell</b>

<h2>Environments Used </h2>

- <b>Windows 10</b>
- <b>Windows 11</b>
